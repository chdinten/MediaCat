Review checklist

Work through each item before approving or rejecting. Notes are optional but encouraged.

Runout / Matrix numbers
  • Matrix A matches label and Discogs entry
  • Matrix B matches label and Discogs entry
  • Hand-etched additions (e.g. "A" / "B" suffix) noted
  • Stamper codes consistent with pressing origin
Media grading
  • Vinyl / disc surface inspected under raking light
  • Grade assigned (Mint / NM / VG+ / VG / G / P)
  • Deep scratches or pressing defects noted
  • Plays without significant noise or skips
Cover & sleeve
  • Front and back cover images uploaded
  • Sleeve grade assigned separately from media
  • Seam splits, ring wear, writing or stickers noted
  • Cut-out or promo markings recorded
Inner sleeve
  • Original inner sleeve present (paper / printed / poly)
  • Condition and type noted (generic / printed / custom)
  • Lyric sheet or insert present if applicable
Inclusions & inserts
  • OBI strip (front / back / spine) — condition and presence noted
  • Hype sticker present on shrink wrap
  • Poster, booklet, or lyric sheet accounted for
  • Certificate of authenticity or numbered card if applicable
Catalogue data
  • Country of origin confirmed against matrix / label
  • Catalogue number matches label and Discogs
  • Label name and pressing plant (manufacturer) verified
  • Discogs / MusicBrainz IDs cross-checked
